## Pre-requisites

- Node.js v14
- NPM v6

To install and use the correct version of Node.js, run:

```sh
nvm install && nvm use
```

Next, install the ECA dependency:

```sh
npm install -g @exlibris/exl-cloudapp-cli
```

then start the development server:

```sh
npm start
```
